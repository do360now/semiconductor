Digital Logic

In addition to storing information such as numbers, the on/off state of components can be used for decision making. Three tests (AND, OR, and NOT) are used to determine the answers to logical questions. The AND test is true only if all input elements are true. The OR is true if any input element is true. The NOT function inverts true to false. When implemented in electronic circuitry, these tests are called logic gates.
